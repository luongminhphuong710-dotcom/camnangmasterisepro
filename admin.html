Quản trị nội dung website

Đăng nhập bằng tài khoản admin riêng để chỉnh sửa dự án, gian hàng, danh mục và tin tức. Khi lưu, CMS sẽ gọi API trên Vercel để commit dữ liệu mới vào GitHub, sau đó Vercel tự deploy lại website.

Token GitHub nằm trong Vercel Người quản trị chỉ cần tài khoản và mật khẩu. GitHub token được lưu trong Environment Variables của Vercel, không xuất hiện trong trình duyệt.
